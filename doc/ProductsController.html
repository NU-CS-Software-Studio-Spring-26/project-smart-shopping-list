      BLOCKED_RETAILER_HOSTS

      Map scraper exceptions to a single user-facing sentence. We deliberately don’t surface raw exception text (DNS errors, “getaddrinfo(3)”, HTTP codes) to end users — those belong in logs, not in a flash banner. Retailers known to block all automated scraping with bot-defence tooling (Cloudflare / Akamai / PerimeterX). We can’t fetch them server-side at all, so we tell the user up front rather than letting them retry forever.

 57   if @manual || @product.name.present?
 58     @manual = true
 59     @product.auto_refresh = false
 60     @product.last_fetch_error = nil
 61     if @product.save
 62       return redirect_to @product, notice: "Product added."
 63     else
 64       return render :new, status: :unprocessable_entity
 65     end
 66   end
 67 
 68   if @product.source_url.blank?
 69     @product.errors.add(:source_url, "can't be blank")
 70     return render :new, status: :unprocessable_entity
 71   end
 72 
 73   # Some retailers (Lululemon, Nordstrom, Sephora, …) block every server-side
 74   # request with bot management, so the fetch can only ever fail. Detect them
 75   # up front and go straight to manual entry — no wasted 5s call, and a calm
 76   # "heads up" naming the retailer instead of a generic error.
 77   if (retailer = PriceScrapers::BlockedSites.label_for(@product.source_url))
 78     return render_manual_fallback(blocked_site_message(retailer))
 79   end
 80 
 81   begin
 82     result = PriceScrapers.fetch(@product.source_url, timeout: 5)
 83   rescue PriceScrapers::Error => e
 84     return render_manual_fallback(
 85       friendly_scrape_error(e) + " You can fill in the product details below to add it manually."
 86     )
 87   end
 88 
 89   @product.name      = result.title.presence || fallback_name_from(@product.source_url)
 90   @product.image_url = result.image_url if result.image_url.present?
 91   @product.stock_status = result.availability if result.availability.present?
 92   @product.source_url = result.resolved_url.presence || @product.source_url
 93 
 94   if @product.save
 95     if result.price.present?
 96       @product.price_records.create!(
 97         price:       result.price,
 98         store_name:  result.store_name,
 99         url:         @product.source_url,
100         recorded_at: result.fetched_at,
101         source:      "scraped"
102       )
103     end
104     @product.update_columns(
105       last_fetched_at: Time.current,
106       last_fetch_error: nil,
107       auto_refresh: true
108     )
109     redirect_to @product, notice: "Product added! We grabbed its details from the page."
110   else
111     render :new, status: :unprocessable_entity
112   end
113 end

13 
14   # The "price_asc/price_desc" sort path uses GROUP BY products.id, which

16   # with the underlying filtered product count so pagination math is right.

19   count_scope = fuzzy_search(count_scope, params[:search]) if params[:search].present?

21   count_scope = count_scope.where("? = ANY(tags)", params[:tag].to_s.downcase) if params[:tag].present?

26   @tags = Current.user.products.pluck(:tags).flatten.uniq.sort
